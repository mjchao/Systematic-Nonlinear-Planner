Project Files Summary
----------------------
Variables.h / Variables.cpp : Defines an object that tracks variable names

Configure.h : Defines symbols associated with the project

Structures.h / Structures.cpp : Structures for predicates, actions, planner, links, threats, ordering constraints, and others. (You must implement the method "adds" for the "Action" structure)

ReadPrint.h / ReadPrint.cpp : Functions for reading in files and printing out files

planner.h / planner.cpp : The plan search function (you implement)

topSort.h : Provides functions for doing topological sort of plans and checking order consistency

main.cpp : The main function, takes command line arguments

test1.txt, test2.txt, test3.txt : Test cases
